# ガイドライン 2.3.2 - パフォーマンス - 正確なメタデータ

## 審査環境情報

- **提出 ID**: 44eb3e0b-9beb-4149-bda2-6400d703f681
- **審査日**: 2025 年 11 月 25 日
- **審査対象バージョン**: 1.0.0
- **ガイドライン**: 2.3.2 - Performance - Accurate Metadata

---

## 問題点

アプリのメタデータが有料コンテンツや機能を参照していますが、追加購入が必要であることが明確に識別されていません。具体的には、アプリの説明文が、このコンテンツにアクセスするために購入が必要であることをユーザーに通知していません。

メタデータで参照されている有料デジタルコンテンツは、ユーザーがアプリに含まれるものと含まれないものを理解できるように、明確にラベル付けする必要があります。

---

## 対応方法

この問題を解決するには、これらの参照を削除するか、有料コンテンツや機能を別途購入が必要であることを明確にマークしてください。
