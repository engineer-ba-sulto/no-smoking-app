# T3-1-0: 「見える化」機能の画面組み込み

## 概要

実装済みのデータリポジトリと成果計算ロジックを利用して、ホーム画面に禁煙の成果（継続時間、我慢した本数、節約金額）を表示する機能を実装します。

## 目的

- ユーザーがアプリを開いた瞬間に、禁煙による具体的な成果を確認できるようにする。
- 禁煙継続のモチベーションを維持・向上させる。
- バックエンドロジックとフロントエンド UI を統合し、アプリケーションのコア機能を完成させる。

## 依存関係

- **依存タスク**: `T1-3`, `T2-2`
- **担当領域**: フロントエンド

## サブチケット

- **T3-1-1**: 成果計算ロジックの呼び出しと結果の取得
- **T3-1-2**: 計算結果の UI コンポーネントへのバインディング
- **T3-1-3**: 継続時間カウンターのリアルタイム更新実装

## 完了条件

- [ ] ホーム画面に禁煙継続時間が表示される。
- [ ] ホーム画面に我慢したタバコの本数が表示される。
- [ ] ホーム画面に節約できた金額が表示される。
- [ ] ホーム画面に延びた寿命が表示される。
- [ ] ホーム画面に次の目標までの期間が表示される。
- [ ] 継続時間はリアルタイムで更新されること。
