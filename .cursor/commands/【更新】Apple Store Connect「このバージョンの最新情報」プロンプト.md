# App Store Connect「このバージョンの最新情報」プロンプト

プロジェクトの機能を確認して、App Store Connect の「このバージョンの最新情報」の原稿を作成して

## ルール

- 4,000 文字以内
- テキストで出力
- 不要な空行は削除
- 新機能・改善点を明確に記載
- ユーザーへの感謝を含める

## 構成

1. バージョン番号の挨拶
2. 新機能の説明（[新機能] タグ付き）
3. 改善点のリスト（[改善] または [その他の改善] タグ付き）
4. 締めの挨拶（感謝 + レビューのお願い）

## 出力例

```text
バージョンX.X.Xをお届けします！

[新機能] 機能名

機能の説明文。

- 機能の詳細1
- 機能の詳細2
- 機能の詳細3

機能の価値を伝えるメッセージ。

[その他の改善]

- アプリ全体のパフォーマンスを向上
- 細かなバグを修正
- 安定性を改善

いつも「歩時間」をご利用いただきありがとうございます。

今後もより良いアプリを目指して改善を続けてまいります。

ご意見・ご要望がございましたら、App Storeのレビューでお聞かせください。

毎日15分、一緒に健康習慣を続けましょう！
```
