# Apple Store Connect スクリーンショット説明文作成プロンプト

## 重要: 画像の確認

**画像が添付されていない場合、以下のメッセージを表示してください：**

```
スクリーンショットの説明文を作成するには、画像の添付が必要です。
Apple Store Connect用のスクリーンショット画像を添付してください。
```

画像が添付されている場合のみ、以下の手順に従って説明文を作成してください。

---

以下の画像の説明を提案してください。Apple Store Connect のスクリーンショットで使いたいです。

## 要件

1. **端的でパッとわかりやすい**: 10-15 文字程度の短いフレーズで核心的な機能を簡潔に表現
2. **文字数制限**: 各説明文は 10-15 文字程度（最大 255 文字以内だが、できるだけ短く）
3. **日本語と英語版**: 両方の言語で説明文を提供（日本語版のみでも可）
4. **視覚的な理解**: スクリーンショットを見ただけで機能が分かる内容
5. **キャッチーさ**: 宣伝文句ではなく、機能の本質を端的に伝える

## 出力形式

各スクリーンショットごとに以下の形式で出力：

```
## スクリーンショット [番号]: [画面名]

**説明文:**
```

[日本語の説明文（10-15 文字程度の端的なフレーズ）]

```

**英語版:**
```

[English description (concise, 10-15 characters)]

```

```

## 参考スタイル

既存の説明文のスタイルを参考にしてください（**端的でパッとわかりやすい**短いフレーズ）：

- ホーム画面: 「禁煙の進捗をリアルタイムで確認」
- 深呼吸タイマー: 「禁断症状を深呼吸で乗り越える」
- 成果・バッジ: 「達成度をバッジで可視化」
- 設定画面: 「あなた専用にカスタマイズ」

**重要**: 長い説明文は避け、10-15 文字程度の短く端的なフレーズで表現してください。

## 背景色の提案

**各スクリーンショットごとに、画像の内容とアプリのテーマカラーを考慮して適切な背景色を提案してください。**

### 提案時の考慮事項

1. **アプリのテーマカラー**: エメラルドグリーン（`#10B981`, `#059669`）を基本とする
2. **画面の内容**: 各スクリーンショットの画面内容に合わせて最適な背景色を選択
3. **可読性**: スクリーンショットの内容が読みやすくなる色を選ぶ
4. **統一感**: 複数のスクリーンショット間で統一感を保つ

### 出力形式

各スクリーンショットごとに以下の形式で背景色を提案：

```
**背景色の提案:**

- **推奨色**: [色名]（`#カラーコード`）
- **グラデーション**: [開始色] → [終了色]（該当する場合）
- **Figma設定**: [具体的な設定方法]
- **理由**: [なぜこの色が適切か]
```

### 参考: アプリのテーマカラー

- **Primary-500**: `#10B981` (RGB: R:16, G:185, B:129)
- **Primary-600**: `#059669` (RGB: R:5, G:150, B:105)
- **Primary-50**: `#f0fdfa` (薄いグリーン)
- **Primary-100**: `#ccfbf1` (薄いグリーン)

## 注意事項

- **端的さを最優先**: 10-15 文字程度の短いフレーズで、パッと見て理解できる内容にする
- 技術的な詳細は不要、ユーザー視点で機能の価値を伝える
- 宣伝文句ではなく、機能の本質を端的に伝える
- スクリーンショットに写っている内容と一致させる
- 必要に応じて実装コードを確認して正確な機能説明を行う
- 長い説明文にならないよう注意（「〜です」「〜ます」などの丁寧語は不要）
